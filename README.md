# Swiggy-Data-Analysis
## Project Objective
The objective of this project is to analyze Swiggy food order data and create an interactive Excel dashboard to gain insights into sales performance, customer ratings, and ordering trends.
The project aims to transform raw data into meaningful visual insights that help understand customer behavior, sales distribution, and regional performance.

## Questions
-What is the total sales revenue generated from food orders?

-What is the average order value per transaction?

-What is the average customer rating across all orders?

-How many total orders and ratings were recorded?

-What are the monthly sales trends?

-Which days of the week generate the highest sales?

-Which states and cities contribute the most sales?

-What is the sales distribution between Veg and Non-Veg items?

-Which top 5 cities generate the highest revenue?

-How do quarterly sales, ratings, and orders compare?
